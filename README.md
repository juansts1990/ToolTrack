# 🔧 ToolTrack

Sistema de gestión de herramientas, préstamos y mantenimientos para empresas de construcción.

## 🚀 Tecnologías

- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Base de datos:** MySQL
- **Autenticación:** JWT + bcrypt

## 📦 Instalación

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Configura las variables de entorno
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## 🗄️ Base de Datos

Importa el archivo `tooltrack_estructura_completa.sql` en MySQL.
